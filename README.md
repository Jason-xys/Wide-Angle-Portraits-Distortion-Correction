# Wide-Angle-Portraits-Distortion-Correction
An implementation of Distortion-Free Wide-Angle Portraits on Camera Phones. YICHANG SHIH, WEI-SHENG LAI, and CHIA-KAI LIANG, Google

# Dependences

  - OpenCV
  - Ceres
  - Face Detection
  - Portrait Segmentation
